## Time scope & retention

| Option | Description | Selected |
|--------|-------------|----------|
| 60s only (no new storage) | Reusa aggregate ring; charts mostram últimos 60s; sem schema novo; ships fast | ✓ |
| 60s hot + 1h coarse RAM ring | 60s precisos + 60×1min RAM ring; sem DB, PERS-03 honrado; bounded memory | |
| 60s hot + persisted long-term roll-ups | Roll-ups 5min/1h em Postgres/ClickHouse para 24h+; bends PERS-03 | |

**User's choice:** 60s only — sem novo storage.
**Notes:** Decisão cascateia em todas as outras: charts ficam em 60 pontos, sem downsampling, sem zoom histórico. Operador que quiser "última hora" fica pra fase futura.
