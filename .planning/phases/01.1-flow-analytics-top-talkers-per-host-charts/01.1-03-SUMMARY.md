## New API Surface

### GET /api/traffic/top20

```
Authentication: session cookie required (401 if absent)
CSRF: GET — skipped by csrfMiddleware design

Response 200:
{
  "items": [
    { "ip": "10.0.0.2", "hostgroup": "corp", "bps": 9000, "pps": 10, "dominant_proto": "udp" },
    { "ip": "10.0.0.3", "hostgroup": null,   "bps": 5000, "pps": 10, "dominant_proto": "udp" }
  ],
  "generated_at": "2026-05-19T13:00:00Z"
}

Empty store: { "items": [], "generated_at": "…" }  (200, not 404)
```

### GET /api/traffic/host/{ip}

```
Authentication: session cookie required (401 if absent)
CSRF: GET — skipped

Validation:
  len({ip}) == 0 or > 64  →  400 { "error": "invalid_param", "param": "ip" }
  netip.ParseAddr fails   →  400 { "error": "invalid_param", "param": "ip" }
  host not in active window → 404 { "error": "not_found" }

Response 200:
{
  "ip": "10.0.0.1",
  "hostgroup": "corp",
  "generated_at": "2026-05-19T13:00:00Z",
  "buckets": [
    { "offset_s": 0,  "pps": 10, "bps": 8000, "pps_udp": 10, "bps_udp": 8000, "pps_icmp": 0, "bps_icmp": 0, "pps_other": 0, "bps_other": 0 },
    { "offset_s": 1,  ... },
    ...
    { "offset_s": 59, ... }
  ]
}
```

## Threat Model — Mitigations in Code

| Threat ID | Mitigation | Implementation |
|---|---|---|
| T-01.1-01 (Auth bypass) | Routes inside `r.Group(requireAuth + csrfMiddleware)` | `server.go:74-77` — both `p.Get` calls inside the group closure |
| T-01.1-02 (Log/canonicalization injection) | Length cap before parse; echo `addr.String()` not raw `chi.URLParam` | `traffic.go:96-104` — `len(ipStr) > maxIPLen` guard + `netip.ParseAddr` + `addr.String()` in response |
| T-01.1-03 (IP enumeration via 404) | Identical 404 for "never seen" and "stale" | `traffic.go:108-111` — single `buckets == nil` branch, no detail |
| T-01.1-05 (XSS via hostgroup) | `encoding/json` HTML-escapes by default; Vue `{{ }}` escaping in Plan 05 | `traffic.go:55-57` — `matches[0].HostgroupName` passed through `json.Encode` |
| T-01.1-06 (Info disclosure via logs) | No `slog.*` calls in handlers; global middleware logs method+URL only | `traffic.go` — no slog imports, no direct log writes |
| T-01.1-07 (CSRF) | GET methods only — csrfMiddleware skips GET by design | `server.go` — only `p.Get(...)` registrations |
| T-01.1-08 (Resource exhaustion) | `maxIPLen = 64` checked before any parsing | `traffic.go:13,97` |
